# Local prompts

This directory is for prompts created by the local Monolith user.

Use this area for:
- machine-specific prompts
- personal workflows
- private homelab prompts
- local model test prompts
- experimental prompt drafts

These files are intentionally not meant to be published as Monolith defaults.

Public-safe prompts should avoid assuming:
- a specific username
- a specific machine name
- a specific operating system
- access to private files or services
- Wumbo-specific homelab context

Suggested layout:

    prompts/local/chat/
    prompts/local/eval/
    prompts/local/system/

Monolith may add deeper UI support for local prompt loading later. For now, this directory documents the intended split between bundled defaults and local/private prompts.
