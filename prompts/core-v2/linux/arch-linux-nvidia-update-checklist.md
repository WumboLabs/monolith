# Arch Linux NVIDIA Update Checklist

Create a conservative Arch Linux update checklist for a desktop with:

- Arch Linux
- NVIDIA GPU
- Wayland desktop
- Local llama.cpp CUDA workloads
- Important working environment that should not be broken casually

Requirements:

- Include pre-update checks.
- Include checking Arch News before updating.
- Use full system upgrade practices only.
- Include NVIDIA/CUDA/driver risk notes.
- Include verification after reboot.
- Include rollback/recovery ideas.
- Avoid risky or novelty recommendations.
- Keep it practical for a real workstation.
